## 🚀 Key Features

<details>
<summary><b>📝 Intelligent Notes</b></summary>
<br>

- Rich Text Editor (`flutter_quill`).
- Support for Bold, Italic, Lists, Checkboxes, etc.
- Auto-save with debouncing.

</details>

<details>
<summary><b>🤖 AI powered</b></summary>
<br>

- Summarize notes using Gemini AI.
- Suggest titles automatically.
- Enhance OCR scanned text.

</details>

<details>
<summary><b>📸 OCR Scanning</b></summary>
<br>

- Capture text from images using ML Kit.
- Real-time crop and preview.
- Seamless conversion to editable notes.

</details>

<details>
<summary><b>⚡ Offline First</b></summary>
<br>

- Local storage with Hive.
- Background synchronization with Firebase.
- Works without internet connection.

</details>

## 🛠️ Tech Stack

<hr style="height:1px;border:none;color:#333;background-color:#333;" />

- **Framework**: `Flutter`
- **Language**: `Dart`
- **Database**: `Hive (Local) / Firestore (Remote) / Firebase Storage (Images)`
- **Cloud Interface**: `Firebase`
- **State Management**: `Bloc / Cubit`

## 💻 Getting Started

<hr style="height:1px;border:none;color:#333;background-color:#333;" />

1. **Clone the repository**
   ```bash
   git clone https://github.com/Korya0/notiva.git
   ```
2. **Install dependencies**
   ```bash
   flutter pub get
   ```
3. **Run the project**
   ```bash
   flutter run
   ```
